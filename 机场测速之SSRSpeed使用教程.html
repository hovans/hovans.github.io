一、基本使用

1.下载软件并解压

下载SSRSpeed，已经停更到2.7.3了。直接下zip文件即可。

https://github.com/hwanz/github_NyanChanMeow_SSRSpeed/releases

2.先安装python，并且把python加到环境变量。这样可以在cmd 命令行在任意一个文件夹输入python就可以调用

3.安装python依赖

到文件夹下，cmd
输入：
pip install -r requirements.txt

4.进入文件夹下，打开cmd

快速在某个文件夹打开cmd方法：

1.点击地址栏，然后输入cmd 回车即可。
2.shift+右键，在此处打卡powershell窗口
然后输入、


回车即可。

5.测完后结果在result文件夹

或者输入


然后浏览器打开：
http://127.0.0.1:10870/
然后就很简单了。

结果
如图：
第一列一般是机场名字；
第二列是节点名称；
第三列是丢包比例；
第四列是到机器的ping情况，一般中转机场就是你到中转服务器时间；
第五列google ping,就是到谷歌服务器的时间了，越低越好。
第六列avgspeed，测速平均速度单位M/s，主要看这个，越大越好。
最后一列是最大速度。
