下面下两种海妖交易所出金路径。

前提:需要有港卡，分别去注册和认证kraken海妖交易所，ifast，wise，中银香港/汇丰/众安银行卡；

都是注册然后认证即可：
kraken海妖交易所注册
ifast注册 点击开户
wise注册 需要入金激活。从中银香港或者汇丰fps转账即可。

把你的u转账到海妖交易所，参考： 币安和欧易okx使用教程

网上很多办卡的教程了。目前办理中银香港需要你的信用卡账单，用途就说是投资，然后也一定要有投资经历，比如支付宝或者银行卡基金理财记录。众安连接香港网络就可以办理。

三种出金路径：

1.转u到海妖交易所kraken pro-> usdt转为GBP英镑（0.2%手续费） -> 提现英镑到ifast(2欧元) -> FPS转账到wise（每天限额5000欧元） -> wise转英镑为港币(0.33%损耗) -> wise到香港卡

走fps，到账快，wise可能风控。

简单一点就是：转 u 到海妖交易所 kraken -> 提现英镑到 ifast(2 欧元) -> FSMOne 香港。需要开fsmone（而且不能线上转账认证，需要线下或者支票，才能连接ifast）

2.转u到海妖交易所kraken pro -> usdt转为美金（0.2%手续费） -> 美金兑换港币（0.48%损耗） ->转账提现到 众安银行（统一33刀手续费）

走swift, 众安可能风控。

中银和汇丰加密收款货币会风控，所以比较麻烦。众安可以直接收

网上主要都是视频教程，东一块西一块，总结下。具体也可以查下

3.不用海妖等交易所，用冷钱包，转到直接到胜利证券

需要用 btc 或 eth 充值后卖出

一、海妖kraken->ifast->wise->港卡

电脑版,下面有手机版:

1.1 海妖交易所pro版本–左边栏交易板块–将usdt转为GBP英镑(0.2%手续费)

pro版本损耗低：https://pro.kraken.com/app/trade/usdt-gbp

1.2 转账英镑到ifast

1.2.1 先去ifast找到你的账号信息: https://www.ifastgb.com/

1.2.2 去海妖交易所非pro版本（pro版本右上角可以切换版本），点击【转账】–【取款】–资产【GBP/Pound Sterling】–方式[Clear junction(FPS)]—【管理取款账户】–填写ifast账户

清算号码就是sort code

1.2.3 转账-取款

1.2.14 确认–2欧手续费

左边【交易】页面看状态：等处理–需要几个小时到账

1.3 ifast通过fps转到wise（每天限额5000欧元）

1.3.1 找到wise账户信息

1.3.2 ifast点击顶部【支付和转账】–英国境内付款–相同币种

https://www.ifastgb.com/payment-and-transfer/payment-within-uk/same_currency

直接复制粘贴wise转账信息：

姓名姓是在后的
收款人Sort Code就是sort 码
收款账号就是wise的【账号】

然后第一次会分笔让你付款。第二次就直接转了。
秒到账的，无损。

手机版本：

1）海妖交易所pro版本app 将usdt转为GBP英镑：中间按钮–搜索USDT/GBP–卖出

2）转账英镑到ifast：底部投资组合–取款–搜gbp–看上面电脑版教程填写资料–下一步。

3）ifast通过fps转到wise：汇款–英国境内汇款–看上面电脑版教程填写资料





1.4 wise到港卡

1.4.1 去wise转换英镑为港币HKD

wise点击英镑余额，然后点击【兑换】即可。会有费用。手续费0.33%

1.4.2 找到港卡信息

中银香港开fps id:

中银香港打开右下角【选单】–【登记转数快】–FPS ID右边【登记】–就是开通FPS的id。


汇丰银行开fps id:

app右上角–点进去人头像–设置和偏好设置–转账和缴费–转数快(注册转数快或者查看转数快信息）-快速支付系统识别码–注册）


1.4.3 放几天后wise转账到中银香港

放几天后再转走，wise据说容易风控
wise点击港币余额–点击【发送】–【自己】–【填写信息】

姓名姓在前 ，名在后，先写收款FPS id会出现你的信息的

1.5 结果

1100usdt ，换为欧元：866.91欧元。ifast到账 864.96欧元。手续费2欧。欧元最后转为港币：8483 港币。费用2.85欧元。

对比币安神盾：1100U，大概是8085人民币。直接换是8630港币。

二、海妖交易所 -> usdt转为美金 -> 转账提现到 众安银行

2.1 海妖交易所 -> usdt转为美金(0.2%手续费)

参考上面的将usdt转为GBP教程：就是把usdt/gbp改为usdt/usd。

2.2 转账提现到众安银行（33刀手续费）

2.2.1 需要先去众安银行获取你的账号信息，众安银行app首页的【账户信息】可见

2.2.2 去海妖交易所普通版本【转账】页面–【取款】–资产选【US dollar】–方式[Bank Frick(swift)]–【管理取款账户】–填写众安银行信息

这里走的不是FPS，而是switf，比较慢，第一次可能需要5天左右。第二次第二天就到账了

2.2.3 众安美金兑换为港币（0.48%损耗）

首页点击【货币兑换】即可。
汇率差。

2.3 结果

1000usdt ifast到账 999.55 刀。到众安卡：964.55 刀。主要是海妖有 2 刀手续费和海妖刀众安有中间固定 33 刀手续费。然后美金转港币还有0.48%损耗。最后转为港币：7475 港币。

众安转出需要去设置限制。

3.冷钱包->胜利证券

不用海妖等交易所，用冷钱包，转到直接到胜利证券

胜利证券认证需要海外地址证明。需要去淘宝买。直接搜海外身份证明。开户流程参考：https://vpsxb.net/7094/
需要转10000港币到这激活颜值。

转到冷钱包。USDT 目前只支持 ERC20 网络；

用 btc 或 eth 充值后卖出。

三、汇款回大陆或者消费

中银香港卡/汇丰 。【汇款】港币回大陆的中国银行，然后【结汇人民币】
