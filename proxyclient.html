Windows安卓Mac和ios代理客户端软件整理推荐

本文最后更新于 2025年5月18日 晚上

整理当前各个系统代理软件客户端，包括windows，mac, ios， 安卓和linux系统，还有路由器的代理软件。主要是包括clash家族的还有xray和singbox家族。最壮大的应该就是clash家族的。桌面端使用较多。ios一般入门都是使用小火箭Shadowrocket。安卓大部分人使用clash for android。很多桌面段的clash gui都有linux版本。
要说各系统推荐什么，建议自己一个个试试，喜欢哪个留哪个：
windows、mac和安卓都推荐clash系列的。小白推荐使用hiddify
IOS小白推荐小火箭、想捣鼓的用QuantumultX去广告；想要免费就推荐使用Potatso、streisand和karing

自用稳定高速国外ssr-v2ray-Trojan专线机场推荐

下面说的全协议：SS/SSr/Trojan/Vmess/Vless/Hysteria2/reality/TUIC,可能少部分。

1) Windows：clash verge rev:支持中文和clash.metan内核; Clash For Windows: 停更了也值得推荐，使用最方便稳定; Hiddify-Next :使用最简单; V2rayN（支持更多协议，很适合自建用户）; nekoray使用教程:类似v2rayN，支持最多协议 ；karing
2) Mac：Clash For Windows和Clash-verge-rev的Mac版本 ; ClashX; ClashX Pro; Surge（贵且不支持ssr）、 sing-box使用教程 ; V2RayU ；karing ； M1、M2还可以使用小火箭
3) IOS：Shadowrocket小火箭：最常见; QuantumultX ：有门槛 ; Stash（IOS版本Clash，使用简单）; Loon（使用简单）; 有钱的买Surge ; sing-box; karing
4) 安卓：ClashforAndroid(CFA、最常见); Surfboard（好看但不支持ssr）; V2rayNG(最大众) ; NekoBox; 支持最多协议 ; sing-box ;karing
5) Linux：Shell clash、Clash For Windows; clash-verg-rev的Linux版本
6) 路由器： OpenWRT固件的Passwall ; OpenClash使用教程

clash系列的、v2rayN、sing-box、Hiddify、NekoBox、Nekoray、shadowsocket小火箭基本啥协议都支持：包括ss、ssr、trojan、vmess、reality、hysteria2等；QuantumultX目前不支持hysteria2和reality这些新的，其它旧的协议基本都支持； 但shadowsocksR软件不支持ss订阅，所以如果你还用着shadowsocksR软件只能买ssr协议的机场，不能买ss协议的； sstap支持ss协议,需要使用订阅转换为SS（SIP002）； Singbox、Surfboard和Surge不支持shadowsocksR协议。
